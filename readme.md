# Priyanshi Singh - Portfolio Website

This is a personal portfolio website created by Priyanshi Singh, a 17-year-old web developer from India.

The website showcases personal information, education, journey in web development, and future goals. It is designed with modern UI styling, smooth animations, and responsive layout.

## Features

- Fixed responsive navigation bar with hamburger menu
- Animated hero section
- Profile card with hover effects
- About section with animated info cards
- Interactive journey section with glassmorphism text boxes
- Smooth hover animations and scroll effects
- Fully responsive design for mobile and desktop

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations, Media Queries)
- JavaScript (DOM manipulation, Intersection Observer API)

## Sections Included

- Hero Section
- About Me Section
- Web Development Journey Section
- Responsive Navigation
- Footer

## Purpose

This project was created as a learning portfolio project to practice front-end development, UI design, animations, and responsive layouts.
